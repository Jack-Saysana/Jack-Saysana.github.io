Project 1 - [OpenGL Graphics Engine]

This is a full graphics/game engine library written in C. It is designed with the intention of providing game/graphics developers with an easy means of loading, simulating and rendering models.

Technologies Used:

  • Written fully in C and OpenGL
  • Utilizes the GLFW, cglm and glad libraries

Features:

  • 3D model loading, instancing and editing workflows
  • Skeletal animation support
  • Collision detection and physical simulation of models
