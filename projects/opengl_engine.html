This is a full graphics/game engine library written in C. It is designed with the intention of providing game/graphics developers with an easy means of loading, simulating and rendering models.

Features:

> 3D Model Management

  • Supports importing fully rigged models with joint and hit-box data
  • Easily import 3D models created in Blender
  • Modify and save model data in code
  • Load and play skeletal animations

> Physics Simulation

  • Fast and robust collision detection of 3D Model Instances
  • Impulse based collision response
  • Articulated body (ragdoll) support

> User Interface Design

  • Rich interface for hierarchical UI design
  • Custom UI texturing
  • Text support with importable fonts
  • On click/hover callback support

> Mathematics

  • 6D Vector Algebra Support
  • Arbitrary matrix and vector arithmetic support
  • Linear system of equations solver

Technologies Used:

  • Written fully in C and OpenGL
  • Utilizes the GLFW, cglm and glad libraries
